列表
无序列表使用星号(*)、加号(+)或者减号(-)作为标记
* 第一项
* 第二项
* 第三项

+ 第一项
+ 第二项
+ 第三项

- 第一项
- 第二项
- 第三项

- 项目一
  * 项目一
    + 项目一
+ 项目二
  - 项目二
  - 项目二
    * 项目二

有序列表直接在文字有加上1. 2. 3. 来表示，符号和文字之间加上一个空格字符，如：
1. 项目1
2. 项目2
3. 项目3

- [ ] 计划任务
- [x] 完成任务
